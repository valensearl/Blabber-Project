📱 Nama Platform: Blabber

🚀 Deskripsi Singkat:

Blabber adalah platform digital yang dirancang untuk memfasilitasi komunikasi dan kolaborasi antar mahasiswa di UIN Salatiga. Dengan fitur-fitur seperti forum diskusi, berbagi materi, dan notifikasi, Blabber membantu mahasiswa tetap terhubung dan produktif dalam lingkungan akademik.

🎯 Masalah yang Diselesaikan:

Blabber membantu mahasiswa dalam berkomunikasi dan berkolaborasi secara efektif.

🧩 User Flow Ringkas (dari User Flow Diagram):

  1. Landing Page -> Registrasi
  2. Home Page -> Memilih fitur platform
  3. Search Page -> Mencari pengguna atau unggahan
  4. Notification Page -> Melihat notifikasi
  5. Profile Page -> Mengelola profil pengguna
